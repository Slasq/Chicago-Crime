# Chicago-Crime
Projekt na zaliczenie przemiotu wielowymiarowa analiza danych
Dane: https://data.cityofchicago.org/Public-Safety/Crimes-2001-to-Present/ijzp-q8t2/about_data
W celu szybszego działania najlepiej przyciąć csv za pomocą:
```bash
Get-Content chicago_crime.csv -TotalCount 1000001 | Set-Content chicago_crime_1mln.csv
```
